# AWS DevOps Challenges

This repository contains a set of hands-on AWS DevOps challenge writeups and supporting artifacts used for learning and demonstration.

Contents
- `challenge-1-meet-your-agent/` — Challenge 1 findings and screenshots
- `challenge-2/` — Challenge 2 findings and screenshots
- `challenge-3/` — Challenge 3 findings and screenshots
- `challenge-4/` — Challenge 4 findings and screenshots
- `challenge-5/` — Challenge 5 findings and screenshots

How to use
- Review each challenge folder for the `Findings.md` (or `FINDINGS.md`) document describing the root cause, mitigation steps, and evidence.
- Screenshots are stored in the `Screenshots/` subfolders for each challenge.

Contributing
- This repository is primarily a record of completed challenges. If you make improvements, please commit clear, small changes and push to the `main` branch or open a pull request.
